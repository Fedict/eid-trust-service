README for eID Trust Service SDK
================================

=== 1. Overview

This ZIP contains the eID Trust Service SDK.


=== 2. Installation

Just unpack the ZIP.


=== 3. Manual

The eID Trust Service Developers Guide can be found in eid-trust-service-devel-guide.pdf.
